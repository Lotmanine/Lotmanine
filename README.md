# Lotfi O. - WordPress Systems Engineer

**CRM migration · Automation · Email deliverability**

I build and integrate the systems behind WordPress and Shopify businesses: custom plugins and themes, CRM/ESP migrations, n8n automations, and email deliverability. Not page builders and templates, the engineering underneath them: data models, access logic, integrations, and the deliverability that decides whether the email lands.

Most of my client work is under NDA, so the repositories below are either built from scratch as reusable tools or adapted from real engagements with all client data, domains, and credentials stripped.

---

## What I work on

- **CRM migration** - moving contacts, lists, automations, custom fields, suppression, and consent provenance between platforms (FluentCRM, ActiveCampaign, Klaviyo, Mailchimp, Keap, Brevo, HubSpot) with post-migration integrity checks so nothing silently drops.
- **WordPress / WooCommerce engineering** - custom plugins, MU-plugins, FluentSnippets, CPTs, taxonomies, access gating, and CRM integrations (LearnDash, Memberium, WooCommerce, ACF).
- **Automation** - n8n workflows and AI-assisted pipelines for lead generation, enrichment, auditing, and back-office process automation.
- **Email deliverability** - SPF/DKIM/DMARC alignment, sender reputation, list hygiene, and Gmail/Yahoo bulk-sender compliance.
